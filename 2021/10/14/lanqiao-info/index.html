第十三届蓝桥杯大赛介绍

发表于 2021-10-14 更新于 2021-10-23 分类于 蓝桥杯 阅读次数：

比赛介绍：

“蓝桥杯”全国软件和信息技术专业人才大赛是由工业和信息化部人才交流中心、教育部全国高等学校学生信息咨询与就业指导中心决定联合举办。据悉，自大赛举办以来，来自全国1200余所高校的10万余名选手报名参赛，其中包括北京大学、清华大学、北京航空航天大学等百余所985、211知名高校也派出了实力选手积极参加大赛，是业界参赛人数最多、影响力最大的学科竞赛。

比赛项目：

  1. C/C++程序设计
  2. Java 软件开发
  3. Python 程序设计
  4. Web应用开发
  5. 视觉艺术设计赛

大赛官网

蓝桥杯赛事：https://dasai.lanqiao.cn/

报名时间

2021年10月14日——2021年12月17日

报名费用

  • 校内模拟赛（自愿）：29.8元/人
  • 省赛：300元/人

比赛时间

  • 校内模拟赛：2021年11月或12月
  • 省赛：2022年3月底或4月

校内交流群
